# Contributing (Solo Dev Standard)

## Branches

- main: production
- dev: staging/integration
- feature/_, hotfix/_

## Commit messages

Use Conventional Commits:

- feat:, fix:, refactor:, chore:, docs:

## Definition of done

- Types clean
- Lint clean
- Build passes
- Server-side validation + role checks for mutations
- Logged mutation (where relevant)
