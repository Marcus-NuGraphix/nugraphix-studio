# ADR-0006: Branding Baseline (Theme + Logos)

- Status: Accepted
- Date: 2026-02-17
- Decision Owners: Nu Graphix (Marcus)
- Related: Phase 2 Design System

## Context

Nu Graphix Studio requires a consistent brand identity across marketing and admin.

## Decision

- Use Nu Graphix brand styling in `src/styles.css`
- Replace starter logos with Nu Graphix company logo assets in `public/` and any UI references
- Maintain cyan/emerald theme alignment with shadcn tokens

## Consequences

- Branding is stable and consistent early
- Future UI work must respect this baseline
