## 3-failure rule

After 3 failed attempts:

- Stop
- Diagnose (top suspects)
- Add targeted logs
- Confirm root cause
- Apply one minimal fix
- Add prevention note to `docs/ai-mistakes.md`

## Production incidents

- Hotfix branch
- Rollback plan
- Postmortem required for S1
